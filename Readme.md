# SmartLearn AI

SmartLearn AI is an AI-powered programming tutor designed to help Indian students understand programming concepts in simple language.

## Features
- AI-powered concept explanation
- Code explanation
- Multi-language support (English, Hindi, Marathi)
- Beginner-friendly interface
- Cloud-based scalable architecture

## Architecture

User → React Frontend → CloudFront → S3 → API Gateway → Lambda → Amazon Bedrock

## Tech Stack
React  
AWS S3  
AWS CloudFront  
API Gateway  
AWS Lambda  
Amazon Bedrock
